Một phiên bản làm lại của trò chơi Rắn Săn Mồi (Snake Game) cổ điển với giao diện người dùng (UI) hiện đại, hiệu ứng kính mờ (Glassmorphism) và khả năng tương thích đa nền tảng.

🎮 **[Chơi thử ngay tại đây!](https://ththth0303.github.io/ran-san-moi/)**

## ✨ Tính năng nổi bật

- **Giao diện Hiện đại:** Thiết kế tối màu (Dark mode) kết hợp với hiệu ứng neon và glassmorphism đẹp mắt.
- **Responsive Design:** Trải nghiệm chơi mượt mà trên cả máy tính (Desktop) và điện thoại di động (Mobile).
- **Đa phương thức điều khiển:**
  - ⌨️ Bàn phím (W, A, S, D hoặc các phím mũi tên) cho PC.
  - 👆 Nút điều khiển ảo (D-pad) cho màn hình cảm ứng.
  - 🖱️ Cử chỉ vuốt (Swipe) trực tiếp trên vùng chơi.
- **Lưu kỷ lục:** Hệ thống tự động lưu trữ điểm cao nhất của bạn ngay trên trình duyệt bằng LocalStorage.
- **Thử thách tăng dần:** Trò chơi sẽ tăng dần tốc độ khi bạn ăn được nhiều mồi hơn.

## 🛠️ Công nghệ sử dụng

Trò chơi được xây dựng hoàn toàn bằng các công nghệ Web tiêu chuẩn, không sử dụng Framework hay thư viện bên thứ ba:

- **HTML5 Canvas:** Dùng để render đồ họa hiệu năng cao.
- **CSS3:** Xử lý layout, tạo các hiệu ứng bóng đổ (glow), kính mờ và gradient.
- **Vanilla JavaScript:** Xử lý logic vòng lặp game, phát hiện va chạm và tương tác người dùng.

## 🎮 Cách chơi

1. Bấm **Bắt Đầu** để vào game.
2. Điều khiển chú rắn ăn các chấm đỏ (thức ăn) phát sáng.
3. Mỗi lần ăn mồi, bạn được cộng 10 điểm và rắn sẽ dài ra thêm một chút.
4. Trò chơi kết thúc (Game Over) nếu đầu rắn chạm vào bất cứ bức tường nào bao quanh hoặc tự cắn vào cơ thể của chính mình.

## 🚀 Cài đặt môi trường máy tính

Vì đây là một trang web tĩnh, bạn không cần cài đặt môi trường máy chủ phức tạp.

1. Clone repository này về máy của bạn:
   ```bash
   git clone https://github.com/ththth0303/ran-san-moi.git
   ```
2. Nháy đúp chuột mở file `index.html` bằng bất kỳ trình duyệt web nào (Google Chrome, Firefox, Safari, Edge,...).
